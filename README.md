CNC driver source prepared for platform NUCLEO-32F439ZI. Project prepared in STM32CubeIDE.

Used pinout:
PE9 - AXE_X_PULSE
PE11 - AXE_X_DIR
PF14 - AXE_X_LIM
PC6 - AXE_Y_PULSE
PB15 - AXE_Y_DIR
PB12 - AXE_Y_LIM
PA6 - AXE_Z_PULSE
PD14 - AXE_Z_DIR
PD15 - AXE_Z_LIM
PB8 - SPINDLE_PULSE
PB9 - SPINDLE_CLOCKWISE
PA5 - SPINDLE_CCLOCKWISE
PB4 - PROBE
PF12 - ESTOP
